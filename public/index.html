Week 1

Environment setup, syntax, core data structures, functions

 Week 3

Pandas: DataFrames & Data Handling

Series, DataFrame, loading data, cleaning, grouping, merging

 Week 4

Visualization: Matplotlib & Seaborn

Line, bar, scatter, histogram, boxplot, subplots, styling, visual EDA

 Week 5

File Formats & Data Pipelines

CSV, JSON, XML, Excel, text parsing, regex, cleaning pipelines

 Week 6

HTTP, APIs & Web Scraping

requests library, REST APIs, JSON handling, BeautifulSoup, ethics

 Week 7

Web Frameworks: Flask & FastAPI

Routing, templates, REST APIs, Pydantic, async basics, deployment concepts

 Week 8

Machine Learning with scikit-learn

ML workflow, preprocessing, classification, regression, pipelines, evaluation

 Week 9

Scientific Python: SciPy & SymPy

Optimization, integration, statistics, symbolic math, equation solving

 Week 10

Practical Statistics for Data Science

Descriptive stats, distributions, correlation, hypothesis testing, confidence intervals

 Week 11

NLP: NLTK, spaCy & Transformers

Tokenization, stemming, NER, POS tagging, text classification, Hugging Face

 Week 12

Computer Vision with OpenCV

Image representation, transformations, filters, edge detection, contours

 Week 13

Databases & Data Engineering

SQLite, SQLAlchemy ORM, ETL pipelines, Dask for large data

os/pathlib, subprocess, selenium, logging, pytest, automation scripts

Packaging, documentation, README, demo, retrospective, deployment concepts
