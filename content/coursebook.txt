Block 1: What is Python &amp; Your First Program
Python is a high-level, interpreted, dynamically-typed language known for its readable syntax. It is the #1 language in Data Science and ML. We start by understanding the Python execution model and writing our first lines of code.
Example:
# Your very first Python programprint(&quot;Hello, Python World!&quot;)# Python as a calculator&gt;&gt;&gt; 2 + 3 * 4      # 14  (PEMDAS applies)&gt;&gt;&gt; 10 / 3         # 3.333... Python uses indentation (4 spaces) to define code blocks, not curly braces. Getting this right is non-negotiable.

Block 2: Variables, Types &amp; String Formatting
Variables are named references to values. Python infers the type automatically. The four primitive types are int, float, str, and bool. F-strings (Python 3.6+) are the modern way to embed values in strings.
Example:
# Variable assignmentname     = &apos;Alice&apos;          # strage      = 30               # intheight   = 5.7              # floatis_adult = True             # bool# Check the typeprint(type(age))            # &lt;class &apos;int&apos;&gt;# F-string formatting (modern, preferred)print(f&quot;{name} is {age} years old and {height:.1f} ft tall.&quot;)# Alice is 30 years old and 5.7 ft tall.# Type conversionage_str = str(age)          # &quot;30&quot;pi      = float(&apos;3.14159&apos;)  # 3.14159

Day 2
Block 3: Control Flow: if / elif / else
Control flow lets your program make decisions. The if/elif/else chain evaluates conditions in order and executes the first matching block. Conditions use comparison operators (==, !=, &lt;, &gt;, &lt;=, &gt;=) and logical operators (and, or, not).
Example:
score = 82if score &gt;= 90:    grade = &quot;A&quot;elif score &gt;= 80:    grade = &quot;B&quot;elif score &gt;= 70:    grade = &quot;C&quot;else:    grade = &quot;F&quot;print(f&quot;Score: {score} → Grade: {grade}&quot;)  # Score: 82 → Grade: B# Ternary (one-liner)result = &quot;pass&quot; if score &gt;= 50 else &quot;fail&quot;

Block 4: Loops: for &amp; while
Iteration is fundamental in data processing. The for loop iterates over any iterable (list, range, string, dict…). The while loop repeats as long as a condition is True. Use break to exit, continue to skip.
Example:
# for loop with rangefor i in range(5):          # 0, 1, 2, 3, 4    print(i, end=&apos; &apos;)       # 0 1 2 3 4# iterating a listfruits = [&apos;apple&apos;, &apos;banana&apos;, &apos;cherry&apos;]for fruit in fruits:    print(f&quot;  - {fruit}&quot;)# enumerate gives index + valuefor idx, fruit in enumerate(fruits, start=1):    print(f&quot;{idx}. {fruit}&quot;)# while loopcount = 0while count &lt; 3:    print(count)    count += 1

A list is an ordered, mutable sequence of items. A tuple is ordered but immutable. Lists are the workhorse data structure for sequential data. Master slicing and list comprehensions — they appear in every real Python codebase.
Example:

Block 6: Dictionaries &amp; Sets
A dictionary stores key-value pairs; lookups are O(1). A set stores unique items with fast membership testing. Both are essential for data aggregation, counting, and deduplication.
Example:

Why NumPy?
Python lists are flexible but slow for numerical computation — they store object references, not contiguous numeric data. NumPy&apos;s ndarray stores typed numeric data in contiguous memory, enabling C-speed computation via vectorization (replacing slow Python loops with a single array operation).

📌 Key Concept
Vectorization: instead of looping over elements in Python, pass the entire array to a NumPy function. This is 10–100x faster and the code is shorter.

Day 1
Block 1: Creating Arrays &amp; Understanding Shape
The fundamental NumPy object is the ndarray. You can create arrays from Python lists, from built-in generators, or by loading files. The shape, ndim, dtype, and size attributes describe the array&apos;s structure.
Example:
import numpy as np# From Python lista = np.array([1, 2, 3, 4, 5])print(a.shape)   # (5,)   — 1-D, 5 elementsprint(a.dtype)   # int64# 2-D array (matrix)m = np.array([[1, 2, 3],              [4, 5, 6]])print(m.shape)   # (2, 3)  — 2 rows, 3 colsprint(m.ndim)    # 2# Built-in constructorsnp.zeros((3, 4))          # 3x4 matrix of 0.0np.ones((2, 2))           # 2x2 matrix of 1.0np.eye(4)                 # 4x4 identity matrixnp.arange(0, 10, 2)       # [0, 2, 4, 6, 8]np.linspace(0, 1, 5)      # [0, .25, .5, .75, 1]np.full((2, 3), 7)        # 2x3 filled with 7

Block 2: Indexing, Slicing &amp; Reshaping
NumPy indexing mirrors Python list indexing but extends to multiple dimensions. Boolean indexing (fancy indexing) is especially powerful for filtering data without explicit loops.
Example:
a = np.array([10, 20, 30, 40, 50])# Basic indexingprint(a[0])      # 10print(a[-1])     # 50print(a[1:4])    # [20 30 40]# 2D indexing [row, col]m = np.arange(12).reshape(3, 4)   # [[0..3],[4..7],[8..11]]print(m[1, 2])   # 6   (row 1, col 2)print(m[:, 1])   # [1, 5, 9]  (all rows, col 1)print(m[0, :])   # [0, 1, 2, 3]  (row 0, all cols)# Boolean indexing — filter without a loopdata = np.array([3, -1, 4, -1, 5, -9, 2, 6])positives = data[data &gt; 0]   # [3, 4, 5, 2, 6]data[data &lt; 0] = 0           # replace negatives with 0# Reshapeflat = np.arange(24)cube = flat.reshape(2, 3, 4)   # 3-D: 2 layers, 3 rows, 4 cols

Day 2
Block 3: Vectorized Operations &amp; Universal Functions
NumPy operations apply element-wise to entire arrays without Python loops. Universal Functions (ufuncs) are pre-compiled C functions that operate element-wise: np.sqrt, np.exp, np.log, np.sin, etc.
Example:
a = np.array([1.0, 4.0, 9.0, 16.0])b = np.array([2.0, 2.0, 3.0, 4.0])# Arithmetic — element-wiseprint(a + b)         # [ 3.  6. 12. 20.]print(a * b)         # [ 2.  8. 27. 64.]print(a / b)         # [0.5  2.  3.  4.]# Scalar broadcastprint(a * 2)         # [ 2.  8. 18. 32.]print(a &gt; 4)         # [F  F  T  T]# Universal functions (ufuncs)print(np.sqrt(a))    # [1. 2. 3. 4.]print(np.log(a))     # [0.   1.386 2.197 2.773]print(np.exp([0,1,2]))  # [1.    2.718  7.389]# Aggregation along axesm = np.arange(12).reshape(3, 4)print(m.sum())       # 66   (total)print(m.sum(axis=0)) # [12 15 18 21]  (column sums)print(m.sum(axis=1)) # [ 6 22 38]     (row sums)print(m.mean(), m.std(), m.max())

Block 4: Broadcasting Rules
Broadcasting allows NumPy to operate on arrays of different shapes by virtually expanding smaller arrays — without copying data. Master the 3 rules: (1) prepend 1s to shorter shape, (2) dimensions of size 1 stretch to match, (3) shapes must be compatible.
Example:
# Rule illustrationa = np.array([[1, 2, 3],    # shape (2, 3)              [4, 5, 6]])b = np.array([10, 20, 30])  # shape (3,) → broadcasts to (2,3)print(a + b)# [[11 22 33]#  [14 25 36]]# Column vector broadcastcol = np.array([[100],      # shape (2, 1)               [200]])print(a + col)# [[101 102 103]#  [204 205 206]]# Practical: center each feature columndata = np.random.randn(100, 5)  # 100 samples, 5 featuresmeans = data.mean(axis=0)        # shape (5,)stds  = data.std(axis=0)normalized = (data - means) / stds   # z-score normalization
🔗 Real-World Connection
Z-score normalization (subtracting mean, dividing by std) is a critical preprocessing step before training ML models. NumPy broadcasting makes it a one-liner.

Block 8: Performance: NumPy vs Pure Python
One of the most important things to understand is HOW MUCH faster NumPy is. This block benchmarks equivalent operations to build intuition for when vectorization matters.
Example:
import timeN = 1_000_000data = list(range(N))arr  = np.arange(N, dtype=float)# Python loopt0 = time.perf_counter()result = [x**2 for x in data]t1 = time.perf_counter()print(f&quot;Python loop:  {(t1-t0)*1000:.1f} ms&quot;)# NumPy vectorizedt0 = time.perf_counter()result = arr ** 2t1 = time.perf_counter()print(f&quot;NumPy vector: {(t1-t0)*1000:.1f} ms&quot;)# Typical output:# Python loop:  320.4 ms# NumPy vector:   1.2 ms   ← 267x faster
⚠️ Common Pitfall
Never loop over NumPy arrays with Python for loops — this destroys performance. Always vectorize. If you cannot, consider numba or Cython.

We build the week&apos;s capstone: a Monte Carlo stock price simulation using Geometric Brownian Motion — a model used in quantitative finance. All computation is vectorized.

# Monte Carlo Stock Price Simulation (GBM)import numpy as nprng = np.random.default_rng(42)S0    = 100.0    # initial pricemu    = 0.10     # annual return (10%)sigma = 0.20     # annual volatility (20%)T     = 1.0      # 1 yearN     = 252      # trading dayspaths = 10_000   # number of simulationsdt = T / N# Random shocks: shape (N, paths)Z = rng.standard_normal((N, paths))# Daily returns via GBM formuladaily_ret = np.exp((mu - 0.5*sigma**2)*dt + sigma*np.sqrt(dt)*Z)# Price paths (cumulative product, prepend S0)price_paths = S0 * np.cumprod(daily_ret, axis=0)price_paths = np.vstack([np.full(paths, S0), price_paths])# Statistics at maturityfinal_prices = price_paths[-1, :]print(f&quot;Mean final price : ${final_prices.mean():.2f}&quot;)print(f&quot;Prob(profit&gt;20%) : {(final_prices &gt; 120).mean():.2%}&quot;)print(f&quot;5th percentile   : ${np.percentile(final_prices, 5):.2f}&quot;)

Day 5: Testing with pytest
pytest is the standard Python testing framework. Tests are plain functions starting with test_. Fixtures provide reusable test dependencies. Parametrize runs one test with multiple inputs.
Key Example:
# test_math.pyimport pytestfrom myapp.math_utils import add, divide# Basic testdef test_add():    assert add(2, 3) == 5    assert add(-1, 1) == 0# Test exceptiondef test_divide_by_zero():    with pytest.raises(ZeroDivisionError):        divide(10, 0)# Parametrize — run same test with different inputs@pytest.mark.parametrize(&apos;a,b,expected&apos;, [    (2, 3, 5), (0, 0, 0), (-1, 1, 0), (100, -50, 50)])def test_add_parametrized(a, b, expected):    assert add(a, b) == expected# Fixture — shared setup@pytest.fixturedef sample_df():    import pandas as pd    return pd.DataFrame({&apos;a&apos;:[1,2,3],&apos;b&apos;:[4,5,6]})def test_dataframe_shape(sample_df):    assert sample_df.shape == (3, 2)# Run: pytest test_math.py -v --tb=short
